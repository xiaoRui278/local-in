# Local-In

局域网P2P聊天工具，基于Tauri + rust-libp2p开发。

## 项目概述

Local-In 是一款跨平台局域网聊天应用，通过P2P技术实现设备间直接通信，无需中心服务器。

### 核心功能

- **局域网自动发现** - 基于mDNS协议自动发现同网段设备
- **P2P聊天** - 点对点加密通信，消息不经过服务器
- **文件传输** - 支持大文件分块传输（开发中）
- **跨平台** - 支持Windows、macOS、Linux

### 技术栈

| 组件 | 技术 | 说明 |
|------|------|------|
| 前端 | React + TypeScript | UI界面 |
| 框架 | Tauri 2 | 桌面应用框架 |
| 后端 | Rust | 业务逻辑 |
| P2P | rust-libp2p | 网络通信 |
| 发现 | mDNS | 局域网节点发现 |
| 加密 | Noise协议 | 传输加密 |

### 开发指南

1. **环境准备**
   - Rust 1.70+
   - Node.js 18+

2. **本地开发**
   ```bash
   pnpm install
   pnpm tauri dev
   ```

3. **打包发布**
   ```bash
   pnpm tauri build
   ```
